座標と方角の見方

→ しがりこソロ鯖マイクラ座標マップ（非公式）に戻る

▲ F3キーで表示された座標と方角の例（XYZとpositive X）

① 座標（X座標 / Y座標 / Z座標）【重要度：★★★★★】

  • 現在位置の座標を表示します。
  • 1つめの数字は X座標（マップのヨコ軸）、3つめの数字は Z座標（マップのタテ軸）を見て移動します。
  • 2つめの数字は Y座標 で高さを示します。移動の時にはあまり使いません。

② 方角【重要度：★】

  • 向いている方角(東西南北)を表示します。
    • east（イースト） → 東（日が昇ってくる方角）
    • west（ウェスト） → 西（日が沈んでいく方角）
    • south（サウス） → 南
    • north（ノース） → 北
  • マイクラでは東西南北よりX/Z座標を見るほうがわかりやすいため、ここは見なくても大丈夫です。

③ X/Z座標の方角【重要度：★★★★】

  • X/Z座標で数字が増える/減る方角を表示します。移動すべき方向が一発でわかります。
    • positive X（ポジティブX） → Xが増える方向（東）
    • negative X（ネガティブX） → Xが減る方向（西）
    • positive Z（ポジティブZ） → Zが増える方向（南）
    • negative Z（ネガティブZ） → Zが減る方向（北）
  • ただし、斜め方向（北東・南東など）もまとめて「ポジティブX（東）」などと表示されます。
  • ブロックにまっすぐ向くと「真東（純粋なポジティブX）」のように正確な方向が分かります。

④ 方角の数字表示【重要度：★★★】

  • 東西南北の360度を－180から＋180までの数字で示しています。慣れるとこの数字と地図で方角調整できます。
  • マップページ「ツール」で「カーソルに追従：方角」を選択するとコンパスが出るので参考にしてください。
    • 北 北西 西 南西 南 南東 東 北東 北
    +180 +135 +90 +45 0 -45 -90 -135 -180

まとめ

  • ここまでの情報を一覧表にまとめると以下のようになります。
    • 北西
    +135    		 negative Z
    北(north)
    ±180    		 北東
    -135
    negative Z
    西(west)日没
    +90    		 あなた positive X
    東(east)日の出
    -90
    南西
    +45    		 positive Z
    南(south)
    0    		 南東
    -45
  • 使い方の例：(-206, -744)から(625, -1780)に移動したい
    • X座標は増える方向、Z座標は減る方向に移動するとよい。
    • positive X とnegative Z の中間方面（つまり北東の-135くらい）に進む。
    • 朝イチ移動なら日の出を右斜め前に見る方角。
    • XかZのどちらかが先に揃うので、次にもう一方を揃える方向を探す。

その他の情報

  • pisitive X などの次の行（Client Light）は明るさを0～15で示しています。7以下は敵が湧きます。
    • カッコ内は内訳で、空からの明るさと光源ブロックからの明るさです。
